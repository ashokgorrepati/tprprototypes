B2C Use Case: Profile-Only (Multi-Profile Query)

This prototype demonstrates a Profile-Only (Multi-Profile) query. This query starts with a dimension from one profile-joined lookup and pulls in metrics from *other* lookups, all joined to a central Person profile.

Persona

A CRM Marketing Analyst.

Use Case

The analyst needs a single, holistic view of their loyalty program. They want to see the total size, lifetime value, and satisfaction score for each loyalty tier. This data lives in separate systems (Loyalty and Survey) but is all linked to the customer profile.

Business Question

"For each of my `Loyalty Tiers`, what is the `Total People` count, their average `Lifetime Value`, and their average `NPS Score`?"

Key Concept: Multi-Profile Pathing
This query is **time-insensitive** (date range is ignored). It works because all lookups (`Loyalty Details`, `Survey Responses`) share a common "root" (`CRM Profiles`). Workspace can "walk" the path from the dimension (`Loyalty Tier`) back to the root profile, and then out to any other joined lookup (`Survey Responses`) to pull metrics.

How This Query Works (The "Path Walking" Logic)

This query is driven by the profile-scoped dimension `Loyalty Tier`. The system calculates each metric by "walking" the join paths:

  1. For "Gold" Tier row:
    • **Dimension (`Loyalty Tier`):** Finds all rows in `Loyalty Details` where `loyalty_tier` is "Gold". (Finds: `L-5001`)
    • `Total People` (Metric 1):
      • Path: `Loyalty Details` (L-5001) → `CRM Profiles` (p-101, p-103)
      • Count(p-101, p-103) = 2
    • `Avg. LTV` (Metric 2):
      • Path: `Loyalty Details` (L-5001)
      • Average(`lifetime_value` where `loyalty_id` = 'L-5001') = $1,200 (from one row)
    • `Avg. NPS Score` (Metric 3):
      • Path: `Loyalty Details` (L-5001) → `CRM Profiles` (p-101, p-103) → `Survey Responses` (S-701, S-703)
      • Average(`NPS_Score` from S-701, S-703) = Average(10, 8) = 9.00
  2. For "Silver" Tier row:
    • Path: `Loyalty Details` (L-5002) → `CRM Profiles` (p-102, p-105) → `Survey Responses` (S-702, S-705)
    • `Total People`: 2
    • `Avg. LTV`: $650
    • `Avg. NPS Score`: Average(7, 8) = 7.50
